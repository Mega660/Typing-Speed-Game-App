# Typing Speed Test Game

An interactive Typing Speed Test Game built using HTML, CSS, and Vanilla JavaScript. This application challenges users to type randomly displayed words within a limited time while tracking their score. The game also includes difficulty levels that affect the time bonus per correct word.
